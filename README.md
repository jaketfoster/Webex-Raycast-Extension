# Webex for Raycast

Interact with Webex directly from Raycast — browse chat rooms, send messages, and manage meetings without leaving your keyboard.

## Commands

### Chat

Browse your Webex rooms with live unread counts and avatars. From here you can:

- Open a room to read messages, view threads, and reply inline
- Send a new message or file to any room
- Edit your own messages
- View and save file attachments
- Call someone directly in the Webex app
- Open any room in the native Webex app

### Meetings

View your upcoming Webex meetings. From here you can:

- Join a meeting in your browser
- Copy a meeting link
- Schedule a new meeting with a title, date, time, and duration
- Start an instant meeting and copy the join link to your clipboard

## Setup

This extension uses OAuth to authenticate with your Webex account. On first launch you will be prompted to sign in via the browser.

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Enter` | Open room / Join meeting |
| `Cmd + N` | Send message / Create meeting |
| `Cmd + D` | Toggle message detail view |
| `Cmd + E` | Edit your message |
| `Cmd + S` | Save attachment |
| `Cmd + Shift + C` | Call person in Webex |
| `Cmd + Shift + F` | Send a file |
| `Cmd + Shift + I` | Start instant meeting |
| `Cmd + Backspace` | Back to all messages (from thread) |
